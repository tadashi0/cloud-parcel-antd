# Cloud Ant Design For Parcel

## 技术栈

- React18
- TypeScript
- Ant Design
- Parcel
- Babel7

## 安装教程

```shell
npm install
```

or

```shell
yarn
```

## 使用说明

```shell
npm run start	//本地环境
npm run build:develop //开发环境
npm run build:release //测试环境
```

or

```shell
yarn start	//本地环境
yarn build:develop //开发环境
yarn build:release //测试环境
```
